# scrumboard

## Stack utilizada

**Front-end:** Angular 18.0 | FuseTheme | TailwindCSS | Material Design

**Moki:** Moki-Api


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/lucsarruda/scrumboard
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  ng serve
```
